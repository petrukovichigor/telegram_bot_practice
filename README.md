# Telegram Bot: Бронирование оборудования (через Google Calendar)

Telegram-бот на базе библиотеки **Aiogram 3**, предназначенный для автоматизации бронирования оборудования в аудитории экономического факультета БГУ. Бот собирает данные через интерактивный опрос (FSM), распознает даты на естественном языке, создает события в **Google Календаре** и мгновенно уведомляет администраторов о новых заявках.

---

## ✨ Основные возможности

- **Интеллектуальное распознавание дат**: Пользователь может писать дату словами (например, `завтра`, `в эту пятницу`, `25 мая`).
- **Гибкий парсинг времени**: Поддерживаются диапазоны (`8:15-11:15`), простые интервалы (`с 10 до 12`) и бронирование по парам (`на 2 пары с 9:00` с автоматическим расчетом длительности).
- **Профили пользователей**: Сохранение ФИО инициатора в локальную базу данных (`user_profiles.json`) для отображения в описании события.
- **Интеграция с Google Calendar API**: 
  - Автоматическое создание события в выбранном календаре.
  - Поддержка регулярных (еженедельных) бронирований.
  - Ссылка на созданное событие возвращается пользователю и админу.
- **Уведомление администраторов**: После успешной брони бот рассылает уведомления со ссылкой на событие всем администраторам, указанным в конфигурации.

│   └── profile.py          # Регистрация ФИО пользователя (/start, /profile)

│   ├── __init__.py         # Создастся автоматически
│   ├── calendar_api.py     # Интеграция с Google Calendar API

│   └── user_profiles.py    # Работа с локальным JSON-файлом профилей
├── .env.example            # Шаблон настроек окружения
├── config.py               # Валидация и чтение переменных окружения
├── credentials.json        # Ключи сервисного аккаунта Google (скрыто)
├── main.py                 # Точка входа для запуска бота
├── requirements.txt        # Список зависимостей проекта
└── user_profiles.json      # Хранилище профилей (скрыто)
